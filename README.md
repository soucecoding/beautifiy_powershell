# beautifiy_powershell
> a code script to beautify windows powershell
> 可以调用一言的API在终端中每次打开即刷新一个句字（网络不好时不建议使用，速度慢一点）。也可以保留刷新的句子，使用命令“yiyan”可以再获取一句新句子并保存。
> You can reset the code script to your own style.
